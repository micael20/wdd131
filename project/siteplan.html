Naval Vessel Guide - Website Plan

Prepared by: Michael Mwidete

Date: December 2024

1. Site Name

Naval Vessel Guide

This name was chosen because it clearly communicates the website's purpose: to serve as an educational guide and reference for various types of naval vessels. The name is professional, descriptive, and easily memorable for users interested in maritime topics.

2. Site Purpose

The Naval Vessel Guide provides a comprehensive educational resource about different types of ships and maritime vessels. The site will offer:

  • Detailed information about various ship classifications (container ships, tankers, naval vessels, etc.)
  • High-quality images and specifications for each vessel type
  • Educational content about naval architecture and marine engineering principles
  • Interactive features for comparing different ship characteristics
  • Accessible maritime knowledge for students, enthusiasts, and professionals

3. Scenarios

Scenario 1: "I'm a high school student working on a maritime history project. What are the main differences between modern container ships and traditional cargo vessels, and where can I find comparison images?"

Scenario 2: "I'm planning a career in marine engineering and want to understand the various types of naval vessels. Can this site show me the different ship classifications with their specific purposes and technical specifications?"

Scenario 3: "I'm a maritime enthusiast visiting a port and saw an unusual ship. How can I identify what type of vessel it is using the resources on this website?"

4. Color Schema

Deep Navy Blue (#003366) - Used for headers, navigation, and primary accents
Muted Gold (#D4AF37) - Used for call-to-action buttons, highlights, and borders
Dark Charcoal Gray (#333333) - Used for body text and paragraphs
Off-White (#FAFAFA) - Used for background and content areas

5. Typography

Primary Fonts:

  • Cinzel (Black) - Used for H1 headers and main titles
  • Cinzel (Bold) - Used for H2 headers and ship names
  • Fauna One (Regular) - Used for body text and paragraphs
